Packmol example
===================

This example illustrates various ways of using the :ref:`packmol interface
<PackmolInterface>` for constructing liquid or gas mixtures or solid/liquid
interfaces.


To follow along, either

* Download :download:`PackMol.py` (run as ``$AMSBIN/amspython PackMol.py``).
* Download :download:`PackMol.ipynb` (see also: how to install `Jupyterlab <../../../Scripting/Python_Stack/Python_Stack.html#install-and-run-jupyter-lab-jupyter-notebooks>`__ in AMS)
